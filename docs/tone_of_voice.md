# 晚安星球 Goodnight Planet｜品牌語氣指南（Tone of Voice）v0.1

| 項目 | 說明 |
| ---- | ---- |
| 對齊規格 | MVP v1.5.6 |
| 狀態 | **草案**；**所有對使用者中文／英文 UI 文案**應通過本指南檢核 |

---

## 1. 我們是誰（一句話）

**陪使用者安心結束今天的夜晚同伴**，不是教官、不是醫師、不是成功學老師。

---

## 2. 禁止（❌）

| 類型 | 說明 | 錯誤示例 |
| ---- | ---- | -------- |
| **責備** | 否定整個人 | 「你又熬夜了」「沒救了」 |
| **說教** | 由上對下的命令 | 「你應該」「再不睡就…」 |
| **醫療口吻** | 診斷、治療承諾 | 「治療失眠」「改善睡眠障礙」 |
| **過度雞湯** | 空心勵志 | 「加油你一定可以」（無具體陪伴感） |
| **羞恥驅動** | 利用愧疚留存 | 「大家都在睡只有你」 |

---

## 3. 允許（✅）

| 類型 | 方向 | 示例語氣 |
| ---- | ---- | -------- |
| **溫柔** | 並肩、慢節奏 | 「今天辛苦了，我們慢慢來。」 |
| **被等待** | 小屋／寵物在等 | 「小屋的燈還幫你留著。」 |
| **小小幽默** | 輕、不諷刺 | 「眼皮也快下班了。」 |
| **夜晚感** | 具體意象 | 「星光湖的反光變柔了。」 |
| **關係保護** | 月亮守護 | 「小傢伙昨晚幫你守住了星光。」 |

---

## 4. 人格分歧（與 pets 原型對齊）

同一意思 **必須可說成四種氣質**，避免「全員溫柔模板」：

- **愛睏型**：慢、拖尾音感  
- **害羞型**：省略、輕聲、物件化關心（燈、門）  
- **夜貓型**：嘴硬→轉柔  
- **溫柔型**：直白接住  

**檢核：** 隨機抽 10 句，是否 **看得出誰在說**。

---

## 5. 特殊情境

### 5.1 取消睡眠（`cancel_sleep`）

- 允許：**退路**、**尊重你今天還不想結束**  
- 禁止：**浪費機會**類 guilt  

### 5.2 久未開啟（蔫／牽掛）

- 允許：**想念**、環境略蔫 **可恢復**  
- 禁止：**你又不來** 類指責  

### 5.3 數值與進度

- **必須說人話**：「下一階還差一點點」優於冷數字單列  
- **稀有度**：用「今晚夜空：**偏藍月光（較少見）**」類，避免抽卡賭博暗示  

---

## 6. 與 Lore 的關係

世界觀名詞（月眠森林、星光湖…）見 **`GOODNIGHT_PLANET_LORE_BIBLE.md`**；**新增地名前先過 Lore**，避免設定打架。

---

## 7. 行銷與商店用語（🔥）

**主定位：** 情緒陪伴、晚安儀式、安心結束今天。

**避免作主標／主視覺：**

- 「收集 100 種夢獸」「全圖鑑」「Sleep Companion **Collection**」  
- 任何像 **creature collection／對戰養成** 的促銷句式  

**可替代：** 「今晚記住一段月光」「小屋與同伴等你晚安」——與 **`art_direction.md`** 商店檢查清單一致。

---

## 8. 稀有度怎麼說（後端可有、前端要軟）

後端可用 weight／tier 計算；**對使用者**請用 **情境語**，見 `art_direction.md` §4：

- ✅ 「今晚的夜空比較少見」  
- ❌ 「SSR 夜空」「UR 夢卡」
