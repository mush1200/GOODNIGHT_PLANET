# 晚安星球｜Moodboard（文字版定向）

對齊 `art_direction.md`、`tone_of_voice.md`、`GOODNIGHT_PLANET_MVP_SPEC.md` §八。

## 情緒關鍵字

陪伴、繪本感、柔光、呼吸節奏、夜晚安心結束今天、**清晰進度**（不抽象到底）。

## 色彩方向（建議 Token，非最終品牌）

| Token | 用途 | 方向 |
| ----- | ---- | ---- |
| `night.deep` | 主背景 | 深藍紫，低對比 |
| `night.glow` | 漸層高光 | 星光／月光邊緣鬆散 |
| `surface.card` | 卡片 | 半透明霧面 |
| `text.primary` | 主文 | 偏暖白，避免刺眼 |
| `text.muted` | 次要 | 灰紫低調 |
| `accent.soft` | 強調 | 淡薰衣草／淡金（非抽卡金閃） |
| `state.attune_low` | 牽掛蔫 | **同一色調**略降飽和／亮度，禁止懲罰黑 |

## 質感關鍵字

繪本、毛線／毛氈、小夜燈、枕頭／睡帽、窗台與夜空漸層——**禁止**怪獸對戰、圖鑑編號、SSR／UR UI（`art_direction.md` §1）。

## 字體與排版（Slice）

- 中文：圓潤易讀優先；標題與內文層級分明。
- 英文（若有）：與中文混排留白舒適；避免過粗競技感。

## 影像参考（非素材）

- 月光穿過窗簾、星點柔焦、紙張紋理。
- **避免**：卡牌框、閃光爆炸、屬性條。

## 動效一句話

慢、漂浮、呼吸式透明度／縮放；開箱用柔光擴散，不用開卡金光。
