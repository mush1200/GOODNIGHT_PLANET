# 晚安星球 Goodnight Planet｜美術與動效方向（Art Direction）v0.1

| 項目 | 說明 |
| ---- | ---- |
| 對齊規格 | MVP v1.5.6 |
| 狀態 | **強制參考**：角色、UI、行銷視覺、動效 review 皆應對照本檔 |

**目的：** 在 **原創 IP** 前提下，維持 **情緒陪伴／睡眠儀式** 一致性，避免美術與 UI **不知不觉滑向「怪獸收集／對戰／圖鑑」語彙**。  
法律上睡眠養成無法被壟斷，但 **具體表現** 會構成風險與品牌混淆；**最大防線是產品自己的一致性**（與 `tone_of_voice.md`、`GOODNIGHT_PLANET_LORE_BIBLE.md` 並讀）。

---

## 1. 禁止方向（❌）

| 類別 | 禁止 | 原因 |
| ---- | ---- | ---- |
| 輪廓／造型 | **怪獸感**、高辨識「寵物對戰」剪影、大頭身圓眼 **演化三段** 聯想 | 接近既有「養成對戰生物」印象 |
| 元素混搭 | 強烈 **動物+元素** 招牌式組合（電火水草符號化） | 易聯想特定收集對戰 IP |
| UI | **圖鑑式**：編號 NO.001、空格「未發現」灰剪影拼滿版 | 像「集齊圖鑑」而非回憶 |
| UI | **卡牌式稀有度**：SSR／UR／閃卡框 | 像抽卡／creature gacha |
| UI | **RPG 屬性條**、刀劍盔甲等戰鬥符號 | 與睡眠療癒定位衝突 |
| 行銷 | 「收集 100 種夢獸」「Sleep Companion **Collection**」為主標 | 滑向 creature collection game |

---

## 2. 允許方向（✅）

| 類別 | 方向 | 示例 |
| ---- | ---- | ---- |
| 質感 | **繪本**、**毛線／毛氈**、**枕頭／睡帽**、**小夜燈** | 柔軟、可抱感，非戰鬥生物 |
| 光線 | **柔光**、邊緣鬆散、低對比夜景 | 避免刺眼高光 |
| 角色 | **軟質小型生物**、簡化五官、安靜表情 | 辨識靠配色與帽子／燈罩造型，非「必殺技臉」 |
| 場景 | **房間、窗台、夜空漸層**、列車「聲音」而非關卡 | 與 Lore 地名呼應即可 |

---

## 3. 動效原則

| 原則 | 說明 |
| ---- | ---- |
| **慢** | 節奏對齊呼吸，避免急促 pop-in |
| **呼吸感** | 輕微縮放、透明度脈動，非彈跳 bounce |
| **漂浮感** | 星塵、窗簾、雲，優於激烈位移 |
| **不刺激** | 避免強閃爍、驚喜爆炸粒子（開箱可用柔光擴散，非開卡金光） |

---

## 4. 「記憶冊」類 UI 的呈現方式（與主規格 §八呼應）

**要像「回憶收藏冊」**，不像圖鑑：

- **標題語**：「今晚記住了：一場偏藍的月光」優於「獲得：藍月夜空 UR」。  
- **條目**：日期 + 一句情境 + 小插圖，**不要** 編號與空格強迫完成。  
- **後端可有 rarity 權重**，**前端展示改為**：「較少見的夜空」「今晚雲層特別薄」等 **敘事 tag**，見 `tone_of_voice.md`。
